## Introduction

I'll show how to make a simple DIY Flappy Bird Game using Arduino. This is a low-cost implementation of the Flappy Bird game :)

## Required components

* 1x Arduino Uno
* 1x Standard LCD (16x2)
* 1x Resistor (1K)
* 1x Buzzer
* 1x Small Push Button Switch
* Jumper wire (as needed).
